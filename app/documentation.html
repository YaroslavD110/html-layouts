Documentation

Main questions:
  • How to change the text on my site?
    All the text needs to be changed between the opening and closing tags
    <tag>YOUR TEXT MUST BE HERE</tag>
    Tags can look like this: <h1></h1>, <h2></h2>, <h3></h3>, <p></p>
  • Where i can change this text?
    Most of the changes occur in the file "index.html"
  • Where should the pictures be?
    All pictures and images must be in "assets/img"
  • How to change values in tag attributes?
    Value which you change must be in ""
    <meta name="description" content="NEW VALUE MUST BE HERE" />
  • How change background image?
    If section background is image this section have attribute style where you can change image name
    <header id="home" class="header" style="background-image:url('assets/img/YOUR_BACKGROUND_IMAGE_NAME.EXTENSION(png/svg/jpg/...));">
Title & meta tags 
						
<meta name="description" content="Collesium - html template" />
<meta name="keywords" content="resume, cv, portfolio, personal, designer, personal resume" />

<!--{ SITE TITLE }-->
<title>My personal web-site</title>

Under line <!--{ SITE TITLE }--> you will see a title tag <title>YOUR SITE TITLE MUST BE HERE</title> which is responsible for the title of your site

<meta name="description" content="YOUR DESCRIPTION MUST BE HERE" /> - this tag is responsible for the description of your site

<meta name="keywords" content="YOUR KEYWORDS MUST BE HERE" /> - this tag is responsible for the keywords of your site

Favicon / site icon

<!--{ FAVICON }--> under it you will see a link tag

<link rel="shortcut icon" type="image/x-icon" href="assets/img/YOUR_ICON_NAME.EXTENSION(png/svg/ico/...)" />
which is responsible for the site icon/favicon of your site. Your icon should be in folder "assets/img/YOUR_ICON_NAME.EXTENSION(png/svg/ico/...)".

Logo

In "FIXED MENU AREA" you will find:

<!--| logo |--> under it you can put in your logo

if you want text logo write your text here:
<h1>YOUR LOGO TEXT MUST BE HERE</h1>

if you want img logo put in there it:
<img src="assets/img/YOUR_LOGO_IMAGE_NAME.EXTENSION(png/svg/ico/...)" alt="Site logo" />

Home text

In "HEADER AREA" you will find:

<!--| Author name |--> under it you can write your name:

<h1>YOUR NAME MUST BE HERE</h1>
if you want change typing text, you must go to "assets/js/common.js", in this file you will see "Typed text settings" and under it is:
var typingTextArray = ["Developer.", "Graphic Designer.", "Freelancer."];
you can change this value or add your value, remember that values should be listed via commas and in ""
